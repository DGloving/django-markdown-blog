 Django Dev Blog

Um motor de blog robusto desenvolvido com **Django**, focado em performance e segurança. Este projeto foi construído aplicando conceitos de cache em memória, segurança de variáveis de ambiente e renderização dinâmica de conteúdo.

## Tecnologias Utilizadas

* **Python 3.x**
* **Django 6.0** (Framework Web)
* **LocMemCache** (Sistema de Cache em Memória)
* **Markdown** (Processamento de texto rico)
* **Python-dotenv** (Segurança e Variáveis de Ambiente)
* **SQLite** (Banco de dados de desenvolvimento)

## Funcionalidades

* **Renderização Markdown:** Escreva posts usando sintaxe Markdown para formatação rápida.
* **URLs Amigáveis (Slugs):** SEO otimizado com slugs automáticos gerados a partir do título.
* **Sistema de Cache:** Implementação de `LocMemCache` para reduzir a carga no banco de dados.
* **Painel Administrativo:** Customizado para gerenciamento eficiente de posts e status.
* **Segurança:** Configuração pronta para produção com separação de chaves secretas.
